# Mini To Do List

Projeto desenvolvido como teste técnico para estágio na empresa MoreResults

## Tecnologias Utilizadas
- C# / ASP.NET Core Web API
- HTML5
- Vue 3 / Nuxt
- Swagger

## Funcionalidades
- Listar tarefas
- Adicionar tarefas
- Remover tarefas

## Como Executar o Programa

### Backend
- Abrir o projeto no Visual Studio
- Executar
- Acessar: https://localhost:7117/swagger

### Frontend
- Abrir a pasta do frontend
- npm install
- npm run dev
